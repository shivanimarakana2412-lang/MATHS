# 📊 Healthcare Statistical Analysis Project

## 🏥 Introduction

Healthcare data analysis plays an important role in understanding patient health patterns and improving medical decision-making.  
With the help of statistics and data science, hidden relationships between health factors such as smoking, diabetes, BMI, age, and glucose levels can be discovered.

This project focuses on performing statistical analysis on a healthcare dataset using Python programming and statistical methods.

---

# 🎯 Project Objectives

The main objectives of this project are:

- To analyze healthcare-related data using Python
- To perform statistical hypothesis testing
- To identify relationships between smoking habits and diseases
- To compare health indicators among different groups
- To calculate confidence intervals for numerical variables
- To apply Z-test, T-test, Chi-Square test, and ANOVA
- To study covariance and correlation between variables

---

 📂 Dataset Description

The dataset contains healthcare records of patients with the following attributes:

- Record ID
- Age Group
- Age
- Weight
- Gender
- Region
- Smoking Status
- Exercise Frequency
- BMI
- Blood Pressure
- Diabetes
- Hypertension
- Cholesterol Level
- Glucose Level
- Visit Date

# 🧠 Statistical Concepts Used

## ✅ Hypothesis Testing

Used to determine whether assumptions about data are statistically significant.

### Example:

- Does smoking affect diabetes?
- Do males and females have different BMI values?

---

## ✅ Confidence Interval

Used to estimate the range in which the true population mean is likely to fall.

---

## ✅ Critical Value and p-value

Used p value to compare with z statistics.

---

## ✅ Z-Test

Used for comparing means when sample size is large.

---

## ✅ T-Test

Used to compare means between two groups.

---

## ✅ Chi-Square Test

Used to check relationships between categorical variables.

---

## ✅ ANOVA Test

Used to compare means among multiple groups.

---

## ✅ Correlation & Covariance

Used to study relationships between continuous variables.
