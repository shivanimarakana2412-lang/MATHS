# 📊 Student Exam Probability Analysis

## 📌 Objective
The objective of this project is to understand and apply basic probability concepts using a student exam dataset. It demonstrates how probability can be used in real-world data analysis with Python.

---

## 📁 Dataset
The dataset used in this project is:
**student_exam_data.xlsx**

It contains:
- Study Hours
- Attendance (%)
- Group Discussion (Yes/No)
- Final Exam Result (Pass/Fail)

---

## 🛠 Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- matplotlib-venn

---

## 📚 Concepts Covered

### 1. Basic Probability
Probability measures the likelihood of an event occurring.

**Formula:**
P(E) = Favorable Outcomes / Total Outcomes

---

### 2. Empirical Probability
Calculated using actual dataset values.

**Example:**
Probability of students passing the exam:
empirical_probability = pass_students / total_students

---

### 3. Theoretical Probability
Based on assumptions of equal outcomes.

**Example:**
P(Pass) = 1 / 2 = 0.5

---

### 4. Random Variable & Probability Distribution
- Converted Pass/Fail into numeric values (Pass = 1, Fail = 0)
- Generated random samples
- Calculated:
  - Probability distribution
  - Mean
  - Variance

---

### 5. Venn Diagram
Used to visualize overlapping conditions:

- Event A: Study Hours > 10
- Event B: Attendance > 80%

Calculated:
- Only A
- Only B
- Both A and B

---

### 6. Contingency Table & Probability Calculations
Used to analyze relationship between variables:

- Group Discussion vs Final Exam Result

Calculated:
- Joint Probability
- Marginal Probability
- Conditional Probability

---

### 7. Understanding Relationships

Bayes Theorem Application
Updating our beliefs based on new evidence. We calculated the probability of a student passing, given they have high attendance.

---

## 🎯 Conclusion
This project helps to:
- Understand basic probability concepts
- Differentiate between empirical and theoretical probability
- Apply probability using real data
- Visualize results using charts and diagrams
